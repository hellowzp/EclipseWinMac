Search Menu

You can trigger the FIND, REPLACE and RENAME command from the Search menu, as well as the "Reset" button ("New" in the menu list). This is useful if you prefer the keybord over the mouse.

More interesting for most users however are the Load and Save options which allow to Save the content of an entire Find form (or Replace form), or to load such data previously saved. This is very useful if you have to execute the same complex regular expression more often.

When asked for a name for the form to save you may indeed use any name or phrase you like. In particular: blanks are allowed.
